# BLD Devs 2.0

---
## How to run

*requirement : python installed* 
<br><br>
for Windows :
```
pip install virtualenv
virtualenv venv
venv\Scripts\activate.ps1
pip install -r packages.txt
```

for Linux & Mac :
```
sudo pip install virtualenv
virtualenv venv
venv\Scripts\activate
pip install -r packages.txt
```
